# AiPro 개발 로드맵 및 미흡사항 관리

최종 업데이트: 2026-07-17

이 문서는 AiPro 프로젝트의 1단계~20단계 진행 상황, 완료 여부, 남은 비율, 미흡한 이유, 다음 작업을 한곳에서 관리하기 위한 기준 문서다.

> 주의: 저장소 검색에서 기존 1단계~20단계 문서가 별도 파일로 확인되지 않아, 현재 코드와 지금까지 합의한 개발 순서를 기준으로 재구성했다. 기존 문서가 발견되면 이 문서와 대조하여 수정한다.

## 상태 표시 기준

- ✅ 완료: 현재 요구 범위에서 구현과 테스트가 끝남
- 🟡 부분 완료: 일부 구현됐지만 실제 운영 또는 연결이 남음
- ⬜ 미구현: 코드가 없거나 설계만 있음
- 🔴 차단: API, 인증정보, 실제 운영환경이 없어서 진행 불가

## 전체 요약

- API 없는 자동매매 코어: 약 70% 완료
- 실시간 PAPER 자동매매: 약 20% 완료
- 실제 주문 자동매매: 약 10% 완료
- 머신러닝: 0% 완료
- 딥러닝: 0% 완료
- 강화학습: 0% 완료
- 24시간 자율운영: 약 10% 완료
- 실전 수익성 검증: 0% 완료

## 변경 기록 규칙

각 기능을 수정할 때 이 문서에 다음을 기록한다.

1. 상태 표시 변경
2. 완료율과 남은 비율 변경
3. 완료된 파일 및 테스트 이름
4. 미흡한 이유 갱신
5. 다음 작업 갱신
6. 마지막 업데이트 날짜 변경

완료율은 코드가 존재한다는 이유만으로 올리지 않는다. 아래 조건을 모두 충족해야 완료로 인정한다.

- 구현 완료
- 단위 테스트 통과
- 통합 테스트 통과
- 실제 실행 흐름에 연결
- 오류 및 복구 시나리오 검증
- 문서 갱신
